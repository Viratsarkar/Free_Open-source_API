## 🚀 How to Use

Getting started with any API in this list is simple! Here's a quick walkthrough:

### Step 1 — Pick an API

Browse the categories above and find an API that matches your project's needs. Look at the **Auth** column to find the easiest one to start with — 🟢 `No Auth` APIs work right in your browser!

### Step 2 — Get Access (if required)

For 🟡 `API Key` APIs:
1. Visit the API's documentation link
2. Sign up for a free account
3. Copy your API key from the dashboard

### Step 3 — Make Your First Request

**Example: Fetch a random dog image (No Auth needed)**
```bash
curl https://dog.ceo/api/breeds/image/random
```

**Example: Get weather data with Open-Meteo (No Auth)**
```bash
curl "https://api.open-meteo.com/v1/forecast?latitude=52.52&longitude=13.41&current=temperature_2m"
```

**Example: Get country info with REST Countries (No Auth)**
```bash
curl "https://restcountries.com/v3.1/name/india"
```

**Example: Using an API key (e.g., OpenWeatherMap)**
```bash
curl "https://api.openweathermap.org/data/2.5/weather?q=London&appid=YOUR_API_KEY"
```

### Step 4 — Parse the Response

Most APIs return **JSON**. Here's how to handle it in common languages:

```javascript
// JavaScript (fetch API)
const res = await fetch('https://dog.ceo/api/breeds/image/random');
const data = await res.json();
console.log(data.message); // URL to dog image
```

```python
# Python (requests library)
import requests
response = requests.get('https://dog.ceo/api/breeds/image/random')
data = response.json()
print(data['message'])  # URL to dog image
```

```bash
# Bash + jq (pretty print JSON)
curl -s "https://restcountries.com/v3.1/name/france" | jq '.[0].capital'
```

> 📌 **Pro Tip:** Use tools like [Postman](https://postman.com), [HTTPie](https://httpie.io), or [Insomnia](https://insomnia.rest) to explore APIs visually before writing code!

## 🤝 Contributing

We love contributions! Whether it's adding a new API, fixing a broken link, or improving documentation — every little bit helps. 🙌

### How to Contribute

1. **Fork** this repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/free-public-apis.git
   ```
3. **Create a new branch:**
   ```bash
   git checkout -b add/your-api-name
   ```
4. **Add your API** following the format below
5. **Commit your changes:**
   ```bash
   git commit -m "Add [API Name] to [Category]"
   ```
6. **Push and open a Pull Request!**

### 📋 API Submission Format

When adding a new API, please include:

```markdown
| **API Name** | Short description of what it does | [🔗 Link](https://api-url.com) | 🟢/🟡/🔵 Auth Type | Free tier details |
```

### ✅ Contribution Guidelines

- ✅ Verify the API is **actually free** (or has a meaningful free tier)
- ✅ Include a working documentation URL
- ✅ Correct auth type: 🟢 No Auth / 🟡 API Key / 🔵 OAuth
- ✅ APIs should return **JSON, XML, or plain text** (no proprietary formats)
- ✅ APIs must be **HTTPS**
- ✅ Place the API in the **correct category**
- ❌ No referral links or sponsored content
- ❌ No APIs that are obviously deprecated or returning 404s

### 🐛 Found a Broken Link?

Please [open an issue](https://github.com/YOUR_USERNAME/free-public-apis/issues/new) with:
- The API name
- The broken URL
- The correct URL (if you know it)
